Positioning and Display Reference

Positioning

Declaration Description
position: static; The default positioning of an element which is left-adjusted
position: relative; Allows you to position an element relative to its default positioning.
position: absolute; Removes the element from the flow of the document causing other elements to ignore its positioning.
position: fixed; Causes the element to stay fixed to its position regardless of user scrolling.
position: sticky; Keeps the element in the document flow even an the user scrolls by sticking to a specified location.
z-index: "value"; Impacts how forward or back an element appears to be. The default is 0 but it will accept all intergers.

Display

Declaration Description
display: inline; Causes an element to fall inline with any element directly before or after it that also has an inline display.
display: block; By default, these elements take up the entire width of the screen even if their content does not.
display: inline-block; Combines the features of inline and block display.
